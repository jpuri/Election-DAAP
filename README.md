# Election DAPP

Election is simple DAPP which allow users to cast votes. It has some rules built on top of it like a user can cast only 1 vote, etc.

#### Tech stack

Ethereum based tech stack for local development.

1. Web3
2. Solidity
3. Truffle
4. Ganache
5. Metamask
